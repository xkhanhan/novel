# 主角卡

> 项目：凡仙途｜主角：林远

## 基础档案
- **姓名**：林远
- **年龄**：开局15岁 → 结丹约120岁 → 元婴约300岁
- **出身**：大夏王朝青云山深处的小村落——青山村
- **身份**：普通农家少年
- **外貌**：瘦高个，因长期劳作皮肤黝黑，眼神有超出年龄的倔强。平凡相貌，丢进人堆找不出来
- **性格**：开局时懵懂淳朴，有韧性但不狡猾。话不多，善于观察，吃了亏会记住

## 核心人设

### 核心欲望
追查灭村真相，找到凶手复仇；复仇之后，活下去并追求长生的道路

### 性格弱点
- **认知局限**：从小在山村长大，对修仙世界一无所知，入世后容易被骗、被利用
- **轻信他人**：本性善良，刚开始难以相信人心可以坏到什么程度
- **底层心态**：因为出身卑微，容易自卑，面对强者时本能退缩
- **执念太重**：认定的事情很难回头，有时会钻牛角尖

### 成长弧线（含道心维度）

| 阶段 | 状态 | 驱动力 | 道心层次 |
|:----|:----|:------|:--------:|
| 第一幕：青山村少年 | 纯真、无知、快乐 | 日常生存 | 无（尚未入道） |
| ↓ 转折点：灭村之夜 | 世界崩塌，仇恨点燃 | — | — |
| 第二幕·第一卷：追杀逃亡 | 愤怒追凶，靠本能活下来 | 复仇 + 求生 | 欲望驱动（第一层） |
| 第二幕·第二卷：初入修仙界 | 开始理解世界规则，被欺骗、被利用、学会算计 | 仍然复仇为主，但开始动摇 | 欲望驱动（第一层）→ 信念萌芽 |
| 第二幕·第三卷：站稳脚跟 | 第一次主动选择，不是为了复仇而是"因为我认为这样是对的" | 复仇 + 信念 | 信念成型（第二层） |

### 核心矛盾
- 想报仇但实力不够 → 借助碎玉变强 → 变强又会暴露位置引来追杀
- 想信任别人但总被背叛 → 孤身太危险，组队又怕被背刺
- 不想变得冷酷无情，但这个世界逼着他变狠
- **最大的矛盾**：他被修仙者毁掉了一切，但他只有成为修仙者才能复仇——他必须变成自己最恨的那种人才能活下去

## 三重问心体系——道心的三次哲学拷问

> 这不是三段独立的"关卡"，是林远一生中三次被迫停下来面对自己的时刻。
> 第一次在结丹——他问自己"我是什么人"；
> 第二次在秘境——他问自己"我走了多远"；
> 第三次在元婴——他问自己"我还走得下去吗"。

### 第一重：结丹突破——「你的丹里装的是什么？」
**发生时间**：筑基后期→结丹初期，约120岁
**叙事定位**：林远在散修生涯中积累够了底蕴，第一次面临"凝聚"的考验。结丹的本质是把筑基期积累的一切压缩成丹——但压缩进去的不只是灵力，还有他这百余年来所有的人生。

**核心问题**：
> 你什么都想带着。但你只有一颗丹。装得下吗？

**哲学内核**：
在结丹之前，林远一直在"积累"——积累灵力、积累经验、积累仇恨、积累 debts、积累关系。他以为变强就是把自己拥有的东西变得更多。但结丹逼他做的第一件事，不是"变多"——是"变少"。
要把所有东西塞进一颗丹里。放不下。必须扔。
但扔什么？哪些该留，哪些该扔？如果扔了，那个东西还算是自己吗？

**林远的答案**：
> "我不是要把所有东西都装进去。"
> "我是要找到那件——没有它我就不再是我的东西。把它装进去，其他的，该扔就扔。"

他找到的那个东西是什么？他没有对任何人说过。但认识他的人，能从他在此后的选择里看出来。

### 第二重：玄霄洞府——「你还记得来时的路吗？」
**发生时间**：结丹初期，秘境第三层幻境
**叙事定位**：结丹问心之后，林远带着答案进入了秘境。但幻境花香不是来考验他的"答案"对不对的——是来考验他那个答案是不是真的。

**核心问题**：
> 你说你找到了"没有它你就不再是你"的东西——你怎么证明它是真的？

**哲学内核**：
结丹的答案是"猜"的——他在突破的那一瞬间做出了选择，但那个选择还没被检验过。幻境就是那个检验。当花香把父母、沈芷柔、所有死境一层一层剥在他面前时，他的回答是发自内心的还是自欺欺人的，一验就知。
**结丹是"凝聚"，幻境是"淬炼"。**

**林远的答案**：
他在幻境里站住了，没有逃避。三重问心（遗憾→来时的路→还能撑多久）全部正面回答，用事实证明了他在结丹时选的那条路没有选错。
（详见[秘境设定-玄霄洞府.md](file:///c:/Users/14277/Documents/trae_projects/novel/凡仙途/设定集/秘境设定-玄霄洞府.md#第三层幻境)）

### 第三重：元婴心魔——（预留）
**发生时间**：结丹后期→元婴初期，约300岁
**叙事定位**：到了这个层面，问题不再是"我是谁"或"我怎么证明"——而是"我走到这一步了，够了吗？"

**待定问题初构**：
> 筑基到结丹，你用了近百年。结丹到元婴，你用了近两百年。你现在三百岁了，和那些元婴老怪差不多的年纪——但你不是他们。

## 能力体系

### 灵根
**杂品灵根**——理论上修炼天赋最差的那一档

### 金手指：神秘碎玉

**来历**：
- 三个月前，林远的父亲在青云山深处打猎时，在一处坍塌的古墓旁捡到一块温润的碎玉
- 父亲不识字，不知其来历，只觉得玉质温润，想留给儿子当护身符
- 他将碎玉缝进林远的棉袄夹层里，打算等入冬时让儿子穿上，一直没来得及说
- 灭村那夜，林远在逃亡中穿上了这件棉袄（母亲临行前塞给他的），被幽冥护法一掌击落山溪时，他在冰冷的河水中挣扎，手无意识地摸到了棉袄夹层里那块温润的玉——他这才发现父亲留给他的遗物

**真实身份**：
- 上古大能"碎玉真人"陨落后，其传承核心分裂成七块碎玉之一
- 七块碎玉各有不同能力，林远得到的这块是其中一块（具体是哪一块，他在第一卷中完全不知道——这也是逐步发现的谜题之一）
- 七玉合一可获完整传承，但具体下落无人知晓

**唯一核心能力：灵气吸纳稳定**

> 碎玉不是加速器，不是外挂。
> 它只有一个作用——让杂品灵根的林远能正常修炼。
> 仅此而已。没有更多了。

- **作用**：碎玉在体内形成一个微弱的"灵力共振场"，帮助稳定灵气吸纳，大幅减少杂品灵根固有的灵气逸散问题
- **效果**：杂品灵根的灵气吸纳效率从"极低"提升到"正常"——和普品灵根一样
- **这不是加速**——不会比普品灵根快，只是让杂灵根拥有了正常修炼的资格
- **到筑基期后需要"喂养"**：练气期碎玉自身能量够用，但到筑基期后共振场不稳定了——需要消耗灵石或妖丹来维持
- **没有灵石/妖丹，碎玉就没用**——修炼效率退回杂品灵根水平

**能力的发现过程（⚠️ 暗线，贯穿第一卷）**：
- 林远不是一开始就知道碎玉有什么用的
- 他是在逃亡过程中逐步摸索、试错、从别人口中打听——才一点点拼凑出碎玉的真正作用
- 详见[力量体系.md](file:///c:/Users/14277/Documents/trae_projects/novel/凡仙途/设定集/力量体系.md)中的「能力发现过程」表格

**代价（双刃剑）**：
1. **持续暴露风险**：碎玉的共振场在运转时会持续散发极其微弱的灵力波动——只要在用就有风险，不是"全力运转才暴露"
2. **玄冥宗的感应**：玄冥宗有专门的感应秘法，在一定范围内能捕捉到碎玉的波动（范围随林远修为提升而扩大）
3. **怀璧之罪**：如果被其他修士发现碎玉的存在，匹夫无罪怀璧其罪
4. **资源依赖（筑基期后）**：到筑基期需要消耗灵石/妖丹维持，这会成为巨大的经济负担

## 核心创作原则：代价法则

### 核心理念
- **没有免费的收获**：林远每一次获得功法、资源、机缘，都必须对应真实的付出——时间、精力、伤势、资源消耗、错过其他机会
- **世界不偏爱任何人**：金手指（碎玉）只是磨平了他与其他修士的起跑线差距，不会让他凭空超越别人。该受的伤、该吃的苦一点不会少
- **用心做事自有回报**：林远的收获来自他比别人更认真、更坚持——比如仔细打扫藏书阁时发现了隐藏的功法，而不是靠运气捡到
- **每一个选择都留有关口**：读者应该能说"如果当时他选了另一边会怎样"——每一个获得背后都对应一个失去的机会成本

### 落地规则
- 获得功法 → 必须在错误路径中选正确的，或付出时间精力去钻研
- 获得灵药/资源 → 必有守护或竞争，消耗自身战力储备才能拿到
- 赢得战斗 → 重伤+消耗底牌（丹药/符箓/法器损坏），惨胜
- 获得信息 → 必须主动去打听/交易/冒险换取，不会凭空送到耳边

## 法宝·功法体系（前三卷）

> 核心原则：获取方式必须多样化——杀敌获得、运气获得、代价购买、做出选择、自己锻造，五种方式轮流使用，不重复套路。

### 一、青霜剑（下品法器——进攻核心）

| 维度 | 内容 |
|:----|:----|
| **品级** | 下品法器 |
| **获取方式** | 杀敌获得 |
| **获取节点** | 第1卷第45章——反杀幽冥护法后，从他的储物袋中获得 |
| **获取代价** | 九死一生——林远差点死在那场反杀战里。重伤躺了三天，底牌耗尽 |
| **外观** | 剑身清冷，长约二尺七寸，剑刃泛淡青色寒光，灌注灵力时浮现一层薄霜 |
| **核心能力** | 灵力附着后对低阶修士有灵力侵蚀效果——被青霜剑伤到的敌人，伤口处会残留冰寒灵力，影响灵力运转 |

| **叙事定位** | 前期唯一的位移/逃命手段，也是林远在战斗中"越级周旋"的底气。用过太多次，以至于后来换了更好的身法也改不掉习惯 |

### 三、玄铁内甲（护体法器——防御核心）

| 维度 | 内容 |
|:----|:----|
| **品级** | 无品级划分（材质特殊：玄铁混合少量寒铁） |
| **获取方式** | 杀敌获得（和青霜剑同一来源） |
| **获取节点** | 第1卷第45章——从幽冥护法尸体上扒下来的，当时幽冥护法穿着这件内甲，林远是从他咽喉处刺入的，内甲完好无损 |
| **获取代价** | 同青霜剑——杀敌获得的好处是"一次战斗拿到两件装备"，坏处是"差点死在那场战斗里" |
| **效果** | 筑基初期以下的正面攻击基本能挡住（打在要害部位也不会致命），筑基初期以上只能减伤（大概减三到四成） |
| **限制** | 覆盖范围只有躯干（胸、背、腹），四肢和头部无法防护；长时间穿着会导致皮肤溃烂（玄铁中的寒铁成分对皮肤有腐蚀性，只能隔衣穿，不能贴身）；到他筑基时这件内甲的防护效果已经不够看了 |
| **成长性** | 无——玄铁就是它的一切。不会因为林远变强而变强，到了筑基期就退居二线了 |
| **叙事定位** | 救了林远至少三次命。第1卷靠它扛住了幽冥护法的临死反扑，第2卷靠它扛住了花娘子的暗算 |

### 四、碎玉共鸣·临时突破（自悟底牌——保命核心）

| 维度 | 内容 |
|:----|:----|
| **品级** | 不属于任何功法体系——是林远自己在绝境中悟出来的本能反应 |
| **获取方式** | 濒死被迫觉醒——不是学来的，是身体被逼到极限后碎玉做出的应激反应 |
| **获取节点** | 第2卷第94-95章——林远被玄冥宗外门执事（筑基初期）逼入绝境，储物袋被打飞，青霜剑折断，玄铁内甲被一掌震裂——他在被击飞的半空中，碎玉自行爆发了一次 |
| **效果** | 灵力强度瞬间提升一个小境界（练气巅峰→短暂拥有筑基初期的攻击力），持续约十息。那十息里他的每一剑都带着碎玉的共振场 |
| **代价矩阵** | 使用完毕后的代价分为三层：①经脉大面积损伤（至少卧床半个月，期间无法修炼）；②碎玉波动剧烈放大（当时就暴露了位置，玄冥宗结丹长老周鹤鸣一个月后亲自出手）；③使用后三天内碎玉会进入"沉寂期"——不产生任何共振场，林远的灵力吸纳效率退回杂灵根本底水平 |
| **成长性** | 有——随着林远对碎玉的理解加深，这个能力会从"失控爆发"进化为"可控爆发"。但控制的代价不会变——每次使用都要付出真实的身体损伤 |
| **叙事定位** | 不是用来翻盘的——是用来"不死"的。每一次使用碎玉共鸣，都意味着林远已经被逼到了"不用就死"的地步 |

### 五、破阵锥·消耗型（一次性法器——辅助核心）

| 维度 | 内容 |
|:----|:----|
| **品级** | 特殊法器（消耗型，用一次就碎） |
| **获取方式** | 付出代价购买——花光了在当时几乎是全部的家当 |
| **获取节点** | 第2卷第58章——林远在落霞集黑市从一个老散修手里买来的 |
| **获取代价** | 三颗中品灵石 + 一株"寒髓草"（他在之前逃亡中挖到的，本来打算自己留着冲击练气中期用的）——这在当时几乎是林远全部积蓄 |
| **效果** | 一次性法器，激活后能强行破开低阶阵法（练气期及部分简单筑基期阵法）的薄弱点。破阵时会爆发剧烈灵力震荡，三百丈内都能感知到 |
| **定位** | 不是常规装备——是为了某个特定目标准备的（第一次进遗迹时差点用上，后来苏青璃精通阵法没用成，但一直留着） |
| **叙事意义** | 代表了林远的一种思维模式——他会为可能出现的困境提前做准备，哪怕付出巨大的代价。这把破阵锥一直用到换了好几件装备都没丢，因为"不知道什么时候就用上了" |

### 六、《筑基丹解》残本·杂灵根炼丹手记（知识型道具）

| 维度 | 内容 |
|:----|:----|
| **获取方式** | 用心做事——因为认真打扫藏经阁的角落，才发现了一个被遗忘的书架底层 |
| **获取节点** | 第3卷第124章——林远在碧落宫藏经阁做杂役时发现 |
| **核心内容** | 数百年前一位杂灵根前辈留下的炼丹笔记。那位前辈穷尽一生，用了四枚筑基丹才筑基成功，他把每一次的失败和成功都记录了下来——"第一枚：全废。药材处理不当，丹炉炸裂。""第二枚：半成。杂质太多，药力不足三成。""第三枚：勉强可用。修正了火候，但灵根所限吸收率极低。""第四枚：成了。我终于知道杂灵根炼丹和正常修士炼丹的区别在哪里了——他们炼的是"药"，我们炼的是"命"。每一枚丹里都要加一味自己的精血进去，让丹药在入腹后能主动和杂灵根的灵力产生共鸣。" |
| **叙事定位** | 为第三卷后期"林远必须自己炼丹补根基"提供理论依据和技术铺垫。也是林远在炼丹这条路上的第一个导师——尽管这位导师已经死了几百年 |

### 七、锻剑术入门·残页（炼器种子）

| 维度 | 内容 |

| **获取方式** | 做选择——在遗迹探索的岔路口，林远选了丹方（给苏青璃），放弃了另一卷炼器残卷 |
| **获取代价** | 选了丹方就没法拿炼器卷。但苏青璃后来把炼器残卷的部分内容抄了一份给他（作为回报——因为林远选的丹方里有一味药对她的修炼路线至关重要） |

| **叙事定位** | 这不是一个"立刻就能用"的收获——是一颗种子。林远在前三卷不会靠它炼出什么厉害的法器，但到了后期，当他在深山老林中无人可依的时候，他会庆幸自己当初拿到了这两页纸 |

### 底牌汇总（真正的底牌——不到绝境不动用的手段）

> 底牌不是装备，是"用了就要付出惨重代价"的最后手段。
> 林远目前真正的底牌只有四张：

| 底牌 | 内容 | 代价 | 使用次数 |
|:---|:----|:----|:-------|
| **小诸天挪移符（残）⭐** | 远古空间遁术符箓残品——激发后随机传送百里之外。**只要激活，就一定走得掉** | 一次性消耗 + 目的地完全不可控（可能传送到更危险的地方） | 从未使用——仅此一张 |
| **破阵锥·过载引爆** | 强行灌入全部灵力引爆破阵锥内部储藏的破阵法力，产生持续性灵力爆燃（覆盖十丈，持续约十息，温度极高，任何筑基级灵力护盾都无法硬扛） | 破阵锥彻底损毁 + 使用者被冲击波震伤（轻则内腑震荡，重则经脉受损） | 从未使用——仅此一次 |
| **碎玉共鸣·爆发** | 灵力短期↑一个小境界 | 经脉损伤+三天沉寂期+暴露位置 | 第2卷末用过1次（濒死觉醒）|
| **自碎玉示警（被动）** | 碎玉在遇到致命威胁时发出预警 | 无主动代价，但预警范围有限 | 被动触发 |

> 这四张底牌的共同特征：用了就没了（或用了就要付出长期代价）。
> 所以林远在战斗中的第一原则永远是——**能用常规手段解决，就绝不动用底牌。**

> **小诸天挪移符（残）与前三者不同——它不是战斗手段，是"故事重置键"。当剧情需要林远从任何必死之局中脱身、开启全新地图时，它就会被激活。这张符不是用来解决问题的，是用来制造更大问题的。**

### 小诸天挪移符（残）·设计详注

**来历**：林远在碧落宫藏经阁当杂役的第三年，无意中发现一本名为《九州风物志》的地理志——这本书的装订比其他书厚了一倍。他拆开书脊的夹层，发现了这张叠成指甲盖大小的符箓。上面刻着他看不懂的上古符文，但那种微弱的空间灵力波动——他在藏经阁浸淫三年，已经能分辨出这不是普通的符箓。

**为什么确定它能用**：他把符箓贴在碎玉上试了一次——碎玉微微一震，符箓上的三条裂纹中有一条亮了一瞬。能共鸣，说明还有灵力残留，还没完全腐朽。

**为什么不用**：①仅此一张，用了就没；②目的地完全随机——可能传到安全地带，也可能传进妖兽巢穴、大能洞府、或者地底岩层中被活埋；③他不确定这符的传送距离——百丈？百里？千里？完全没概念。

**叙事定位**：
- 这是林远的**终极保险**——当故事需要他"从任何必死之局中脱身"时使用
- 但传送目的地的随机性意味着——**它解决一个问题，同时制造一个新问题**
- 在读者层面：这张符的存在让读者对林远的处境有"安全感上限"——哪怕被金丹修士堵住了，读者也知道林远还有一张终极牌没打
- 但它的代价（随机传送+永久消耗）又让读者不能完全安心

**叙事规则**：
- 整个第4卷都不能用——太早了，性价比不够
- 第5卷从中期开始进入"有概率使用"的阶段
- 一旦使用，**必定触发新地图/新剧情线**——不能用来"从A逃到A隔壁"

## 禁止事项（防崩坏）
- 林远不会突然变聪明——每个成长都必须对应一次吃亏经历
- 不会在前期获得超越境界的战力
- 碎玉不会突然解锁超出当前需求的能力
- 林远不会随便信任人——但也不会变成冷血无情的杀人机器
- **碎玉不会帮助理解功法**——它只稳定灵气吸纳，不翻译、不教学、不灌顶
- **完整功法是修炼入门的前提**——林远的每一次修炼突破都必须建立在拥有完整功法文本的基础上，片段心法不能无师自通
