# 傀儡体系设计

---

## 总纲：傀儡术的定位

傀儡术在修仙界是一门**偏门但不可或缺**的技艺。

它不是主流的战斗方式——没有哪个修士会把自己的身家性命完全托付给一具机关造物。但在特定的场景下，傀儡有着修士肉身无法替代的优势：它不会累、不怕死、不会背叛、不会因为道心动摇而战力衰退。

傀儡术的繁荣度与修仙界的稳定程度成反比——乱世中傀儡师的数量会急剧增加，因为没有人信得过活人。

但在当今的越国修仙界，傀儡术已经衰落了三百年。原因有二：
- 上古傀儡术的核心传承断代，现存的傀儡术都是残篇，最多只能炼制出筑基中后期水准的傀儡
- 材料成本高昂且不可回收——一个傀儡战损了，投入的灵石和材料就全打了水漂

所以傀儡术在当下的定位是：**大宗门用来守山门、守遗迹的工具手段；散修玩不起，也不值得玩。**

## 傀儡的核心构成

每一具傀儡都由三个核心部分组成：

### 一、躯壳

躯壳是傀儡的身体，决定其基础属性和耐久度。

| 材料类型 | 特性 | 适用场景 | 成本 |
|:--------:|:----|:--------|:----:|
| 凡木/竹 | 轻便但不耐用 | 一次性消耗品、侦查型 | 极低 |
| 青岗岩 | 坚硬但笨重 | 守门/阵地战 | 低 |
| 精铁 | 均衡，可塑性强 | 通用型 | 中 |
| 玄铁/陨铁 | 极高硬度，可承载高阶符文 | 战斗型 | 高 |
| 灵木（千年铁木等） | 轻便且坚韧，可自行缓慢吸纳灵气 | 高阶傀儡 | 极高 |
| 妖兽骨骼/甲壳 | 继承妖兽的部分属性（如速度、防御） | 特化型 | 视妖兽等级而定 |

**原则**：躯壳材料的品质决定了傀儡的**上限**——用青岗岩造的傀儡，无论符文核心多强，也不可能快过用灵木造的。

### 二、符文核心

符文核心是傀儡的大脑和心脏，是其最关键的部件。

- **作用**：储存傀儡的行动指令、战斗逻辑、灵力运转路径
- **材质**：通常用玉简或特制的灵石晶板为载体，以精金丝刻录符文回路
- **工作原理**：符文核心接收灵力输入 → 按预设指令驱动傀儡的各部分机关 → 执行行动
- **重要限制**：符文核心的复杂度决定了傀儡的"智力"上限。低阶傀儡的符文核心只能存储简单的指令（"守卫此区域，排除一切进入者"），高阶傀儡可以存储多层判断逻辑

**符文核心的品质决定了傀儡的"智能"水平**：

| 品质 | 逻辑能力 | 典型案例 |
|:----:|:--------|:--------|
| 粗糙核心 | 仅能执行单一指令（直行/攻击/停止） | 最低级的机关人偶 |
| 普通核心 | 可执行条件判断（"如果……就……"），能分辨敌我 | 青石洞的守护傀儡 |
| 精良核心 | 可执行多层条件判断和简单策略（优先攻击威胁最大的目标，灵力不足时退守） | 大宗门护山傀儡 |
| 极品核心 | 接近低等灵智，可自主制定战术、学习对手的战斗模式 | 上古遗迹中留存的高级傀儡 |

### 三、动力源

傀儡需要灵力驱动才能行动。灵力来源有三种：

| 动力类型 | 工作原理 | 续航 | 优点 | 缺点 |
|:--------:|:--------|:----|:----|:----|
| **灵石驱动** | 在傀儡体内安装灵石槽，放入灵石提供灵力 | 视灵石品质而定（下品灵石约可维持低强度行动数个时辰） | 补充方便，任何人都可操作 | 需要持续消耗灵石，长期成本高 |
| **外部灌注** | 由傀儡师持续向傀儡输送灵力 | 取决于傀儡师的灵力储备 | 可灵活控制傀儡的行动强度 | 占用傀儡师的大量精力，无法同时操控多具 |
| **自循环核心** | 符文核心中刻录了微型聚灵阵，可自动吸纳天地灵气维持运转 | 理论上永久，但战斗时吸收速度跟不上消耗 | 零维护成本 | 炼制难度极高，技术已失传 |

**常见组合**：
- 低端傀儡：普通躯壳 + 粗糙核心 + 灵石驱动
- 中端傀儡：精铁躯壳 + 普通核心 + 灵石驱动/外部灌注
- 高端傀儡：灵木/妖兽骨 + 精良核心 + 灵石驱动为主，自循环核心为辅
- 上古遗物：极品材料 + 极品核心 + 自循环核心（已不可复制）

## 傀儡的品级

与修仙界的通用分级对齐：

| 品级 | 战力对应 | 特征 |
|:----:|:--------|:----|
| **普通傀儡** | 练气初中期 | 躯壳为凡木或粗糙石材，只能执行简单指令，反应迟钝，破绽明显 |
| **中品傀儡** | 练气后期～筑基初期 | 精铁躯壳，可执行多层指令，具备基本的敌我识别能力 |
| **上品傀儡** | 筑基中后期 | 优质材料 + 精良核心，可执行战术策略，能配合修士协同作战 |
| **极品傀儡** | 金丹期及以上 | 极品材料 + 极品核心，接近灵智，可自主进化战术，极其稀有 |

## 傀儡的分类

### 按用途

| 类型 | 说明 | 典型特征 |
|:----|:----|:--------|
| **战斗傀儡** | 正面作战 | 重甲、高耐久、配备近战或远程武器 |
| **守卫傀儡** | 固定区域防御 | 不主动追击，但在其领地内战力加成，通常与阵法联动 |
| **侦查傀儡** | 侦察/探路 | 体型小，速度快，灵力波动微弱，不易被察觉 |
| **工程傀儡** | 采矿/搬运/建造 | 力量大，精度低，无战斗能力 |
| **机关人偶** | 日常服务/装饰 | 无战斗价值，仅用于表演或简单劳作 |

### 按操控方式

| 方式 | 说明 | 代表人物 |
|:----|:----|:--------|
| **预设指令** | 傀儡根据刻录在符文核心中的指令自主行动，不需要操控者实时干预 | 青石洞的守护傀儡属于此类 |
| **灵力丝线** | 傀儡师以灵力凝成细丝连接傀儡的各关节，实时操控其动作。操控精度极高但消耗巨大，且灵力丝线可被切断 | 传统傀儡师 |
| **神魂烙印** | 傀儡师在符文核心中留下自己的神魂印记，傀儡可以理解操控者的"意图"而非"指令"。操控距离极远，但神魂反噬风险极大——傀儡被毁会伤及操控者的神魂 | 上古高阶傀儡师 |

## 傀儡师

### 傀儡师的定位

傀儡师不是一种独立的修炼体系——它是在传统修炼路径上额外选修的一门技艺。

一个修士要成为傀儡师，需要具备以下条件：
- **神识足够强**（至少筑基期）——操控傀儡需要持续以神识监控其状态，神识太弱的话连傀儡在哪都感知不到
- **熟悉符文刻录**——至少能看懂和理解基础符文回路，否则连修傀儡都做不到
- **有足够的灵石**——傀儡是烧钱的活，每一场战斗都在烧灵石

### 现存已知的傀儡相关功法

| 功法名称 | 品级 | 说明 |
|:--------|:----:|:----|
| 《千丝傀儡诀》 | 中品术法 | 以灵力凝丝操控傀儡的手法，最多可同时操控三具。越国散修界流传最广的傀儡术残篇，缺失了核心的"神魂烙印"部分 |
| 《百机诀》 | 上品术法 | 上古傀儡术的完整章节之一，可同时操控九具傀儡，且具备简单的协同作战阵法。但传承已断，现存的部分只有理论描述，无人能真正炼成 |
| 《魂偶术》 | 极品术法 | 将自身神魂分出一缕植入傀儡核心，使傀儡获得近似灵智的战斗能力。此术对神魂损伤极大，每分出一缕神魂都需要数十年才能恢复。此为上古秘术，当世已无人掌握 |

### 为什么要强调傀儡术已经衰落

这是为了控制故事的力量天花板——如果傀儡术太强、太普及，很多剧情矛盾就不成立了。

具体来说：
- 如果傀儡术传承完整，碧落宫就不会被一夜灭门（用傀儡守山门至少能撑到援军到来）
- 如果傀儡术不衰落，林远的很多以弱胜强的战斗就不合理（对方放一具金丹傀儡出来他就没法打了）
- 傀儡术的衰落也是世界观的一部分——修仙界不是一直在进步，而是在某些领域是在倒退的

## 傀儡在故事中的运用

### Vol 2：青石洞守护傀儡

已在[古修士遗迹-青石洞.md](file:///c:/Users/14277/Documents/trae_projects/novel/凡仙途/设定集/古修士遗迹-青石洞.md)中详细设计。

定位：中品守卫傀儡，用青岗岩和精铁打造，筑基中期战力。是林远第一次接触傀儡术，他在这场战斗中学会了"找弱点的能力是可以迁移的"——这个认知会在玄霄洞府中帮到他。

### 后续可能的傀儡线

1. **林远不会成为傀儡师**——他的修炼资源和精力都不允许他去钻研一门辅助技艺。但他会在后续的遗迹探索中多次遇到傀儡——这是"前人的遗产"最典型的守护形式。

2. **季无咎可能和傀儡术产生关联**——季无咎的散修出身和什么都想试试的性格，让他对傀儡术产生兴趣是合理的。但这只是一个可选的支线，不是必须展开的。

3. **玄霄洞府（Vol 5）中的高阶傀儡**——作为上品甚至接近极品的上古傀儡，与青石洞的粗制石傀儡形成鲜明对比，让读者直观感受到"上古传承"和"现代残篇"之间的差距。

4. **傀儡术与碧落宫的关系**——碧落宫旧籍中藏有青石老人的手稿残页，说明碧落宫历史上确实有过傀儡术的研究。碧落宫覆灭后，这些研究成果是否落入了玄冥宗之手？这是一个可以埋下的伏笔。

## 傀儡战损的处理原则

- **傀儡不是一次性消耗品**——哪怕是低端傀儡，在战场上受损后也有修复的可能。傀儡师会回收还能用的部件
- **核心符文如果完整，傀儡就还能救**——躯壳碎了可以换，核心符文碎了傀儡就彻底报废了
- **林远对付傀儡的核心策略**：他不跟傀儡正面硬碰硬，他的打法是找弱点、找漏洞、找指令逻辑上的缺陷。这是他作为智斗型主角的优势
