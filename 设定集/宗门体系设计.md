# 宗门体系设计

> 项目：凡仙途｜创建：2026-05-07｜更新：2026-05-07

## 总纲：修仙界的势力格局

越国修仙界的格局可以用一句话概括：**正道遮天，魔道裂土，中立通吃，散修吃土。**

表面上正道联盟占据道义制高点，魔道联盟龟缩南疆苦寒之地，但真实情况远比这复杂——正魔两道之间利益交织了上千年，各大宗门之间既有明面上的针锋相对，也有暗地里的利益输送。所谓正邪之分，在底层散修看来不过是大宗门垄断资源的遮羞布。

六股势力构成越国修仙界的基本盘：

| 势力 | 定位 | 总部 | 实力评级 |
|:----|:-----|:----|:--------:|
| **天剑宗** | 正道之首 | 中州·天剑山 | ★★★★★ |
| **碧落宫** | 中立·丹阵双绝 | 越国·碧落山脉 | ★★★★ |
| **丹霞谷** | 商业·中立 | 中州·丹霞城 | ★★★★ |
| **幽冥殿** | 魔道之首 | 南疆·九幽山 | ★★★★★ |
| **血煞门** | 魔道·鹰派 | 南疆·血焰谷 | ★★★★ |
| **玄冥宗** | 隐藏反派 | 越国·冥狱山脉 | ★★★☆（表面）→ ★★★★★（真实） |

## 一、天剑宗 —— 正道之首

### 一句话定位

> 天下剑修的圣地，但圣地从来不是好待的地方。

### 宗门概况

| 维度 | 内容 |
|:----|:------|
| **立派时间** | 约三千年 |
| **山门** | 中州·天剑山——七座剑形山峰直插云霄，主峰「问剑峰」顶端常年云雾缭绕 |
| **势力范围** | 中州核心区域，掌控三条上品灵脉 |
| **修士数量** | 内门弟子约三百人，外门弟子约两千人 |
| **最高战力** | 太上长老（元婴后期，半步化神） |
| **核心功法** | 《天剑诀》（极品功法，金属性） |

### 内部结构

天剑宗内部并非铁板一块，分为三大派系：

**剑阁派（主流派）**
- 掌门一脉，占据半数以上长老席位
- 主张剑修正统——剑就是一切，其他术法丹器都是旁门
- 作风高傲但不算恶劣，对中小宗门有"正道之首的架子"但不会主动欺凌
- 与幽冥殿有宿仇，千年来正魔大战的主力

**守旧派（隐退派）**
- 由剑阁守阁人、藏经阁长老等老一辈构成
- 不过问宗门日常事务，但在重大决定上有一票否决权
- 陆沉舟上玄天宗的扫地老人就属于这一类——表面上是扫地老人，实则是上一代的剑阁守阁人
- 这支派系的真实底蕴连掌门都不完全清楚

**革新派（少壮派）**
- 以年轻一辈金丹长老为主，约占总长老席位的三成
- 主张修正道不应固步自封，应该吸纳丹霞谷的商业手段、学习碧落宫的丹药技艺
- 与剑阁派有路线之争，但尚未激化到内斗的程度

### 功法特色

天剑宗的核心功法体系围绕"剑"展开，几乎所有术法都以剑为载体：

- **《天剑诀》**：极品功法，金属性。修至大成可凝聚"本命剑元"，一剑破万法。天剑宗镇派功法，非核心弟子不传
- **《万剑归宗》**：上品术法，标志性大范围攻击术法——以剑元分化万千剑气，覆盖式打击。极其耗费灵力，金丹以下用一次就虚脱
- **《养剑术》**：天剑宗独门秘术。弟子结丹后会炼制自己的本命飞剑，以丹田剑元日夜温养。养得越久，剑越通灵，威力越大。天剑宗的本命剑比其他宗门同阶飞剑强三成以上——因为他们从筑基期就开始养剑
- **《剑遁》**：中品术法，以剑驭身的高速飞行术，速度在同阶遁术中排前三

### 关键人物

**掌门·秦无霜**
- 元婴中期，男，三百余岁
- 面相清癯，不苟言笑，持一柄古剑"霜寒"
- 为人正直但不迂腐——他知道正道联盟有龌龊，但只要不触及天剑宗的底线，他睁一只眼闭一只眼
- 对碧落宫覆灭的态度：**震惊，但不会为一个二流宗门大动干戈**

**太上长老**
- 元婴后期，六百余岁，常年闭关冲击化神
- 姓甚名谁已经没几个人记得了，宗门上下只称"太上"
- 近百年没出过关，传言他的气血已经开始衰败——如果三十年内不能化神，寿元就到头了
- 天剑宗的定海神针，他活着，就没有人敢打天剑宗的主意

**剑阁守阁老人**
- 名号不详，天剑宗上一代传奇
- 就是当年一招折断陆沉舟"斩业"刀的那位扫地老人
- 表面身份是打扫剑阁的杂役老人，实际是上一代剑阁守阁人——当年那件事之后他就卸任了，但人没走，留在了剑阁扫地
- 知道一些三千年来天剑宗和整个修仙界的秘密，但不说不问不管

### 与主线的关系

- 天剑宗在中后期才会正式登场（第7卷以后）
- 林远在秘境（第5卷）中可能遇到天剑宗的弟子——双方因争夺机缘产生冲突，但不会是生死大敌
- 天剑宗对碧落宫覆灭的态度：**不会主动为碧落宫复仇**，但如果发现幕后黑手与正道联盟有关，可能会介入调查
- 剑阁守阁老人的秘密——那条牵涉碧落宫覆灭和玄天宗的伏线，会在后期与林远的路径交叉

## 二、碧落宫 —— 丹阵双绝的中立派

### 一句话定位

> 一个以丹药和阵法立足的二流宗门，不争不抢，却在风暴来临时第一个被碾碎。

### 宗门概况

| 维度 | 内容 |
|:----|:------|
| **立派时间** | 约一千五百年 |
| **山门** | 越国·碧落山脉——九座青峰环抱一座天湖，山水相映，灵气充沛但灵脉品级不高 |
| **势力范围** | 越国中部，掌控两条中品灵脉 |
| **修士数量** | 内门弟子约一百五十人，外门弟子约六百人，杂役不计 |
| **最高战力** | 宫主（元婴中期） |
| **核心功法** | 《碧落心经》（上品功法，水木双属性）、《碧落丹诀》（中品术法）、《落英剑法》（中品术法） |

### 内部结构

碧落宫的特色是**以女修为主但不排斥男修**——约六成弟子为女性，高层中女性占绝对主导。宗门氛围相对温和，但这不代表没有内斗：

**宫主一脉（温和派）**
- 现任宫主及其嫡系，掌握决策权
- 主张保持中立，不参与正魔纷争，专注丹药和阵法的传承
- 对弟子较为宽容——只要不背叛宗门，一般不会重罚
- 对林远的态度：**只要他不惹事，杂役就杂役，没人会注意他**

**外事堂（实权派）**
- 负责宗门对外事务——采购灵草、销售丹药、与外界的利益往来
- 实权极大，掌握着宗门的财路
- **陈长老**（筑基巅峰）是外事堂的实权人物，为人精明世故，手伸得很长
- 外事堂对林远的态度：**有用就用，没用就扔**——陈长老后期因为林远偷丹方而追杀他，不是因为林远得罪了碧落宫，是他拿了外事堂想私吞的东西

**丹堂（技术派）**
- 负责炼丹和丹方研究，地位超然但实权不大
- 青石老人就是被丹堂边缘化后才出走建洞府的
- 丹堂长老们大多痴迷炼丹不谙世事，对外事堂的所作所为睁一只眼闭一只眼

### 功法特色

碧落宫的功法体系侧重**持续作战和生存能力**，而非爆发力：

- **《碧落心经》**：上品功法，水木双属性。灵力绵长浑厚，恢复速度在同阶功法中排名前列——打消耗战有天然优势。缺点是缺乏强力攻击手段
- **《碧落丹诀》**：中品术法，炼丹专用。以特殊手法提炼药材精华，成丹率和品质略高于普通炼丹手法
- **《落英剑法》**：中品术法，水属性。剑势如落花飘零，轻盈灵动，杀伤力一般但变化多端

### 关键人物

**宫主·云霓裳**
- 元婴中期，女，两百余岁
- 面相端庄温和，待人宽厚，深得弟子爱戴
- 但**过于温和**——她明知道外事堂有猫腻也没有严查，明知道玄冥宗在打探消息也没有警觉
- 碧落宫覆灭那夜，她是第一个死的——不是战死的，是被内鬼从背后刺穿丹田

**外事堂·陈长老**
- 筑基巅峰，男，一百五十余岁
- 碧落宫第3~4卷的次级反派，林远的主要追杀者之一
- 精明、世故、自私。他追杀林远不是为了碧落宫的面子——是因为林远从丹房偷走的固本培元丹核心材料，是他私吞了准备拿出去卖的
- 他对外打着"叛徒追杀"的旗号，实际上是要灭口

**杂役·青禾**
- 碧落宫最低等的杂役弟子，没有灵根，在伙房蒸馒头
- 第3卷"安稳的诱惑"的核心角色
- 详见总纲·诱惑成长体系

### 与主线的关系

- 第3卷核心舞台——林远在此潜伏、当杂役、结识苏青璃和青禾
- 第4卷前期——林远被外事堂陈长老追杀
- **碧落宫覆灭事件**（第3卷高潮）——一夜之间从上到下被屠戮殆尽，宫主元婴被内鬼偷袭，具体经过详见「碧落宫覆灭事件」设定
- 覆灭后碧落宫的余部四散，部分弟子逃到散修联盟或投入其他宗门
- 苏青璃是碧落宫覆灭后唯一知道部分真相的人——她看到了内鬼的脸

## 三、丹霞谷 —— 只认灵石不认人

### 一句话定位

> 修仙界最大的商号——不站队、不打架、不惹事，谁给灵石就给谁炼丹。

### 宗门概况

| 维度 | 内容 |
|:----|:------|
| **立派时间** | 约两千年 |
| **总部** | 中州·丹霞城——一座以商业为核心的修仙城市，不设山门，整座城就是丹霞谷 |
| **分布** | 五大洲皆有分号，商路遍布修仙界 |
| **修士数量** | 炼丹师约五百人，商队护卫约一千人，遍布各地的掌柜和伙计不计其数 |
| **最高战力** | 谷主（元婴中期，同时也是修仙界第一炼丹师） |
| **核心理念** | **丹霞三不**——不参与正魔之争、不掺和宗门仇杀、不赊账 |

### 内部结构

丹霞谷不是传统意义上的修仙宗门，更像一个**商业帝国**：

**谷主·丹阳子**
- 元婴中期，男，四百余岁
- 修仙界公认的第一炼丹师——他炼出来的极品丹药，连天剑宗太上长老都要给面子
- 他的信条只有一条：**丹霞谷存在的意义是赚钱，不是打架。赚钱是为了收集更好的材料，收集材料是为了炼更好的丹。至于谁买了丹拿去做什么，不关丹霞谷的事**
- 精明到了骨子里，但有一个致命的弱点：对珍稀丹方的痴迷——如果有人拿失传的古丹方跟他做交易，溢价多少他都接

**各分号掌柜**
- 最低金丹初期，最高金丹后期
- 每个掌柜负责一个区域的生意，拥有极大的自主权
- 掌柜之间也存在竞争——业绩好的掌柜能拿到丹霞城的"分红"，业绩差的会被调去蛮荒地区的分号

**护卫队**
- 丹霞谷的武装力量，以雇佣的散修为主
- 负责护送商队、保护分号安全
- 护卫队的战力不弱——但他们拿钱办事，不会为丹霞谷拼命。真遇到不可力敌的敌人，丹霞谷的原则是"货可以丢，人要活着回来"

### 功法特色

丹霞谷不以战斗见长，但有其独特的辅助功法：

- **《丹霞诀》**：中品功法，火属性。专为炼丹优化的功法——灵力输出极其稳定精纯，适合长时间炼丹操作，但灵力总量和攻击力在同阶中偏下
- **《百转炼丹术》**：丹霞谷秘传炼丹手法，可同时处理多份药材，大幅提升成丹率
- **镇谷之宝·百转炼丹炉**：中品法宝，可同时炼制多份药材，成丹率比普通丹炉高三成

### 与主线的关系

- 丹霞谷在中后期登场（第5卷秘境之后）
- 林远在逃亡中可能需要通过丹霞谷的分号购买丹药或出售战利品
- 丹霞谷不参与正魔纷争的特性，使它成为唯一一个能让正魔两道修士坐下来交易的地方
- 林远后期可能用一个古丹方与丹阳子做一笔交易——换取关键情报或庇护
- 丹霞谷在碧落宫覆灭后的态度：**照常收购碧落宫弟子流出的丹药，不问来源**

## 四、幽冥殿 —— 魔道之首

### 一句话定位

> 魔道不是一群疯子——他们是一群在弱肉强食的规则下活了几百年的人。

### 宗门概况

| 维度 | 内容 |
|:----|:------|
| **立派时间** | 约两千八百年 |
| **总部** | 南疆·九幽山——一座深入地下的阴寒洞窟群，终年不见阳光 |
| **势力范围** | 南疆核心区域，掌控两条上品灵脉（在南疆深处的幽冥分坛还有一条） |
| **修士数量** | 内门弟子约两百人，外门弟子约一千五百人 |
| **最高战力** | 殿主（元婴后期） |
| **核心功法** | 《九幽玄功》（极品功法，暗属性）、《噬魂诀》（上品秘术） |

### 内部结构

幽冥殿与天剑宗不同——**他们没有派系。不是不想分，是不敢分。**

殿主是绝对的独裁者。他的规矩只有一条：**谁的拳头大，谁说了算。** 下面的长老之间可以有竞争、有仇怨、甚至可以斗法——但只要殿主开口，所有人必须闭嘴。

这种统治方式的优点是执行力极强——一旦殿主下令，整个幽冥殿会像一台精密机器一样运转。缺点是殿主一旦出问题，幽冥殿立刻就会分裂。

### 功法特色

幽冥殿的功法体系以"阴""寒""暗"为核心：

- **《九幽玄功》**：极品功法，暗属性。灵力阴寒入骨，侵染力极强——与《天剑诀》硬拼，九幽玄功的阴寒灵力会顺着剑元侵入对方经脉，造成持续伤害。幽冥殿核心功法，只有殿主嫡系才有资格修炼
- **《噬魂诀》**：上品秘术，可直接吞噬他人魂魄碎片来强化自身神识。但副作用极大：每用一次都会导致自身魂魄不稳，长期使用会精神分裂。幽冥殿底层弟子不准修炼——怕他们走火入魔变成疯子
- **《幽冥掌》**：中品术法，中掌者五脏六腑逐渐冻结。玄冥宗的"幽冥掌"就是脱胎于这一门——但威力不及原版的三成

### 与主线的关系

- 幽冥殿在中后期正式登场（第7~8卷）
- 幽冥殿和玄冥宗之间**没有直接关系**——幽冥殿不知道玄冥宗在收集碎玉，玄冥宗也不敢让幽冥殿知道
- 幽冥殿在碧落宫覆灭事件中：**不是他们干的，但会被正道联盟怀疑**——正魔两道都会默认"肯定是魔道干的"
- 林远与幽冥殿的接触：大概率是在南疆的剧情中（第8卷），林远与幽冥殿的关系是**利益交锋而非正邪对抗**——幽冥殿不会因为他是正道散修就追杀他，只会因为挡了他们的路才动手

## 五、血煞门 —— 魔道的疯狗

### 一句话定位

> 魔道中的魔道——在正道看来幽冥殿只是对手，血煞门才是疯子。

### 宗门概况

| 维度 | 内容 |
|:----|:------|
| **立派时间** | 约一千八百年 |
| **总部** | 南疆·血焰谷——一座终年弥漫血腥味的火山峡谷，土壤被精血浸透呈暗红色 |
| **势力范围** | 南疆边缘区域，与十万大山接壤 |
| **修士数量** | 内门弟子约一百人，外门弟子约八百人 |
| **最高战力** | 门主（元婴初期） |
| **核心特色** | 血炼之法——以精血祭炼功法、法宝、丹药，威力大但反噬强 |

### 内部结构

血煞门是六大门派中最"野蛮"的一个——**弱肉强食在血煞门不是潜规则，是明文规定。**

- 每年有一次"血炼大比"——弟子之间不限生死地斗法，最后站着的人才有资格得到门主的指点
- 长老之间一言不合就斗法，门主不管
- 背叛宗门的惩罚是"血祭"——被绑在血焰谷的祭坛上，被门主当众炼成一枚血丹

但血煞门的战力不容小觑——他们的功法虽然邪门，但实战能力在同等境界中往往比正道修士高半档。因为他们不怕死、不惜命。

### 功法特色

- **《血煞大法》**：上品功法，血属性。以自身精血为引驱动术法，威力极大但每次使用都会损耗寿元。血煞门主修功法
- **《血遁》**：中品术法，以燃烧精血为代价换取瞬间远遁。逃命手段第一——用过一次减寿十年。血煞门修士打不过就跑，跑得了就回来报复，跑不了就自爆丹田拉人垫背
- **镇门之宝·血灵刃**：下品法宝，以修士精血喂养的邪刃，杀伤力极大但每用一次都会反噬主人

### 与主线的关系

- 血煞门出现的频率很低——他们主要在十万大山活动
- 第2卷中散修遇到的"吸血陷阱"（花娘子的故事）就是血煞门的低级功法流出的后果——血煞门的外围势力在散修中传播这些残篇，目的是收集散修的精血反馈给总部
- 血煞门与玄冥宗：**有暗中交易**——玄冥宗用灵石和功法向血煞门购买过"血祭秘术"，用于碎玉的激活实验
- 血煞门在中后期（第8卷后）会成为一个次要威胁，但不是主要对手——他们更像"这个世界的黑暗面"的背景色

## 六、玄冥宗 —— 隐藏的暗流

*注：玄冥宗的详细设定已在[反派设计.md](file:///c:/Users/14277/Documents/trae_projects/novel/凡仙途/设定集/反派设计.md)中完整展开，此处仅做补充性概述和与其他宗门的关系梳理。*

### 一句话定位

> 一个表面上只是中等宗门的势力——直到你发现它暗中经营了上千年。

### 宗门概况

| 维度 | 内容 |
|:----|:------|
| **表面定位** | 越国修仙界的中等宗门，不引人注目 |
| **实际实力** | 远超表面——宗主（元婴后期→伪化神），大长老（元婴初期）暗中坐镇 |
| **总部** | 越国·冥狱山脉——终年笼罩在灰色雾瘴中，外人不得靠近 |
| **真实目的** | 搜寻散落在天地间的七块碎玉，重现上古传承 |
| **行事风格** | 极其隐秘，斩草除根，不留活口 |

### 与其他宗门的关系

| 宗门 | 玄冥宗的态度 | 对方的认知 |
|:----|:-----------|:----------|
| **天剑宗** | 避免接触，绝不招惹 | "越国那个小宗门？没什么印象。" |
| **碧落宫** | 利用外部弟子渗透 → 最终覆灭碧落宫（原因见「碧落宫覆灭事件」） | "中等宗门，低调但不值得注意"——直到被灭门 |

| **血煞门** | 有暗中交易——购买血祭秘术用于碎玉激活 | "大主顾，不问来历，灵石给够就行" |

### 关于玄天宗的说明

玄天宗是**中州的隐世宗门**，独立于越国修仙界的正魔格局之外。陆沉舟断刀的故事发生在此。玄天宗的扫地老人身怀"扫尘诀"——这门功法与碧落宫覆灭之间存在关联。玄天宗的秘密将在故事后期（第8卷后）与主线交织。

玄天宗不属于六大宗门体系中的任何一个——它是一个更古老、更隐秘的存在。

## 七、宗门实力对比总览

### 全书元婴分布（按大洲）

元婴是修仙界明面上的天花板——跨过这道门槛，才有资格在棋桌上落子。每一个元婴修士都是有名有姓的人物，没有路人甲元婴。

一个洲的元婴上限：后期≤2人，中期≈3人，初期≈5人，合计≈8人（按最顶级的洲算）。

```
东洲（越国所在）—— 明面元婴：0人（实际隐藏3人）
│
├─ 元婴后期：玄冥宗宗主（隐藏，不为外界所知）
├─ 元婴中期：碧落宫主·云霓裳（已故）
└─ 元婴初期：玄冥宗大长老（隐藏，不为外界所知）
│
└─ 越国地处东洲偏远之地，明面上连一位元婴都没有。
   碧落宫主一死，越国修仙界在高端层面已是真空。
   玄冥宗的两位元婴全部藏于暗处，不到最后不会暴露。

中洲（修仙核心）—— 元婴：8人
│
├─ 元婴后期：天剑宗太上长老（气血已衰）、天枢府首座（正值巅峰） ← 2人
├─ 元婴中期：天剑宗掌门·秦无霜、丹霞谷主·丹阳子、天枢府·副座 ← 3人
└─ 元婴初期：天枢府·法阵长老、天枢府·执法长老、中州散修（1~2人） ← 3~4人
│
└─ 中州是全书元婴最密集的大洲，即便如此也只有8人。
   天剑宗和天枢府包揽了大部分名额。

南疆（魔道地盘）—— 元婴：7人
│
├─ 元婴后期：幽冥殿殿主（巅峰期）、南疆妖王（最接近化神的妖王） ← 2人
├─ 元婴中期：妖王级·青蛟王 ← 1人

苍玄洲 —— 破碎大陆上没有活着的元婴势力，只有遗迹和死物。
```

### 战力评估说明

1. **中州>南疆>东洲**——顶级洲（中州）元婴总量控制在8人以内，这已经是修仙界一万年来积累的全部家底。每一个元婴陨落都是不可逆的损失
2. **天剑宗**的综合实力略胜幽冥殿——不是太上长老比殿主强，而是剑修的中层战力（金丹、筑基）在同阶单挑中有天然优势
3. **幽冥殿**恐怖在殿主一人——元婴后期巅峰、正值壮年、心狠手辣。左右护法配合殿主多年，三人的合击之术足以对抗元婴中期的对手
4. **碧落宫**覆灭的根本原因：全宫仅宫主一位元婴。她一死，碧落宫就是待宰的羔羊
5. **玄冥宗**的真实实力远超表面——宗主是隐藏的元婴后期，大长老是隐藏的元婴初期。但他们不敢暴露，因为一旦暴露就会被正道和魔道同时盯上
