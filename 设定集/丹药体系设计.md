## 一、丹药品级体系

### 品质划分

丹药按品质分为五等：

| 品质 | 特征 | 效果 |
|:---:|:----|:----|
| **普通** | 粗炼而成，杂质多，有丹毒积累风险 | 药力一般，长期服用必留隐患 |
| **中品** | 精炼而成，杂质少，药效良好 | 散修和普通弟子的首选，性价比最高 |
| **上品** | 高水准炼制，几乎无杂质 | 药力优异，有价无市，拍卖行级别 |
| **极品** | 完美炼制，完全无杂质 | 可遇不可求，一枚难求 |
| **灵丹** | 通灵性的丹药，可自主吸纳灵气保存药效 | 传说级别，几百年不一定出一枚 |

品质由两个因素共同决定：
- **材料年份**——百年灵草炼出来的底子比十年灵草好，上限更高
- **炼丹师水平**——大师出手保底中品，学徒出手大概率普通

**示例**：
- 普通筑基丹：药力一般，有丹毒积累风险
- 上品筑基丹：药力优异，几乎无杂质
- 极品筑基丹：药力饱满，完全无丹毒，可能还有额外效果

### 核心设定

- 丹药的品质由**材料年份**和**炼丹师水平**共同决定——百年筑基草+大师出手才有可能出上品
- 年份不足的灵草，神仙来了也炼不出上品——材料的底子决定了丹药的品质上限
- 丹毒：所有丹药都含杂质（丹毒），品质越低丹毒越多。长期服用普通品质丹药会在体内积累丹毒，轻则修为停滞，重则经脉堵塞
- 上品丹药几乎无丹毒，极品丹药完全无丹毒——这就是上品丹价格是普通丹十倍的原因
- 灵丹有丹灵，会自己跑——需要用玉盒封印

## 二、炼丹基本原理

### 炼丹三要素

| 要素 | 说明 | 替代方案 |
|:----|:----|:--------|
| **丹方** | 丹药的配方和炼制步骤，包含药材比例、处理顺序、火候曲线 | 无完整丹方几乎不可能炼出有效丹药 |
| **丹炉** | 提供炼丹所需的封闭空间和温度控制 | 可以用普通鼎/铁锅替代（效果极差，成功率极低） |
| **火焰** | 加热丹炉、熔炼药材 | 普通炭火→地火→修士丹火（随修为递增） |

### 炼丹步骤

```
选材 → 处理（清洗/研磨/浸泡/炮制）→ 入炉 → 控火（升温/恒温/降温）→ 凝丹 → 开炉 → 养丹（高阶丹药需要）
```

每一步的细节决定丹药品质：
- 处理不当→杂质多（品质下降）
- 控火不稳→药力流失（药效降低或废丹）
- 凝丹时机不对→药性冲突（轻则无效，重则炸炉）

### 杂灵根炼丹的特殊性

根据《筑基丹解》残本的记载，杂灵根炼丹与正常修士有本质区别：

- 正常修士炼丹：用丹火+灵力精准控制每一步，追求的是**纯净**
- 杂灵根炼丹：无法精细控制丹火，需要用**精血**在丹药中种下"共鸣引子"——让丹药入腹后主动和杂灵根的灵力产生共振，逼出药力

**代价**：
- 每次炼丹要消耗自身精血（不是普通血液，是蕴含灵力本源的精血）
- 精血损耗后需要至少半个月才能恢复，期间修为停滞甚至轻微倒退
- 连续炼丹会导致严重的气血亏虚——所以那位前辈"四枚筑基丹炼了七年"

## 三、炼丹师等级

| 等级 | 修为要求 | 可炼丹药品质 | 地位 |
|:----|:--------|:------------|:----|
| **炼丹学徒** | 练气中期以上 | 普通品质 | 各大宗门最底层的杂役，成功率极低 |
| **初级炼丹师** | 练气后期以上 | 普通～中品 | 有一定地位，散修中的香饽饽 |
| **中级炼丹师** | 筑基初期以上 | 中品～上品 | 宗门争抢的人才，月俸丰厚 |
| **高级炼丹师** | 筑基后期以上 | 上品 | 一方豪强的座上宾 |
| **炼丹大师** | 结丹期 | 上品～极品 | 整个修仙界凤毛麟角，各宗门长老级 |
| **炼丹宗师** | 元婴期 | 极品～灵丹 | 传说级，一丹难求 |

**设定说明**：
- 修为不是硬门槛——但丹火质量随修为提升而提升，高品质丹药需要更好的火力掌控
- 更多是靠天赋和经验——有些炼丹大师修为不高但手法出神入化
- 苏青璃在练气后期已是初级炼丹师水平，在同龄人中算有天赋的
- 林远没有炼丹天赋——他的优势是"足够认真"（对应主角卡的用心做事路线）

## 四、核心丹药设计

### 筑基丹

| 维度 | 内容 |
|:----|:----|
| **品质** | 普通～上品 |
| **核心作用** | 辅助练气巅峰修士冲击筑基。药力冲击丹田，帮助灵力液化 |
| **正常效果** | 一枚上品筑基丹，普品灵根修士筑基成功率约七成 |
| **林远的情况** | 杂品灵根，一枚普通筑基丹药力只够丹田液化三成——所以需要多枚，或者更高品质的丹药 |
| **材料** | 主料：百年筑基草、地髓根；辅料：凝露花、赤芝、灵泉水 |
| **市场价格** | 普通品质：80～120下品灵石；中品品质：300～500下品灵石；上品品质：1000+下品灵石（有价无市） |
| **获取方式** | 大宗门内部配额 → 黑市交易 → 自己炼丹（林远的路线） |
| **叙事定位** | 林远在Vol 2获得第一枚（苏青璃所赠），Vol 3服用时一枚药力不够——碎玉补足缺口成功突破。这次突破暴露了碎玉位置，直接导致周鹤鸣追猎启动；同时暴露了林远最大的短板：杂灵根需要外部辅助才能突破大境界，这驱动了他Vol 4自学炼丹 |

### 固本培元丹

| 维度 | 内容 |
|:----|:----|
| **品质** | 普通～上品 |
| **核心作用** | 修复根基损伤、稳固丹田。对因强行突破或经脉受损导致的根基不稳有修复效果 |
| **林远的需求** | Vol 3突破时碎玉补足药力成功筑基，根基正常，但三年潜伏期无法正常修炼、经脉长期处于压制状态，导致根基出现自然松动。Vol 4自学炼出一枚普通固本培元丹后成功稳固至筑基中期——损伤不严重，一枚够用 |
| **材料** | 核心材料：培元果、百年黄精、地脉芝；辅助材料：归元草、灵泉水等（从碧落宫丹房偷了三份核心材料，缺辅助材料） |
| **市场价格** | 普通品质：500～800下品灵石；中品品质：1500+下品灵石 |
| **获取方式** | 只能买或者自己炼——林远走了后者 |
| **叙事定位** | Vol 3末～Vol 4的核心驱动力。林远带着三份核心材料逃亡，需要自己学会炼丹、补全辅助材料 |

### 聚气丹

| 维度 | 内容 |
|:----|:----|
| **品质** | 普通～上品 |
| **核心作用** | 加速灵气吸纳，日常修炼用。服用后十二个时辰内灵气吸纳速度提升约三成 |
| **受众** | 练气期～筑基初期修士的日常消耗品 |
| **市场价格** | 普通品质：3～5下品灵石/瓶（每瓶十枚）；中品品质：20～30下品灵石/瓶 |
| **备注** | 碧落宫杂役每月发一瓶最低档聚气丹——是散修眼中的好福利，但对筑基期杯水车薪 |

## 五、灵草分类与生长特性

### 按年份划分

| 等级 | 年份 | 可炼丹药品质上限 | 价值 |
|:----|:----|:--------------:|:----|
| 普通灵草 | 10年以下 | 普通 | 低廉 |
| 百年灵草 | 100～500年 | 普通～中品 | 筑基期丹药主料 |
| 千年灵草 | 1000年以上 | 上品 | 结丹期丹药主料，极其稀有 |
| 万年灵药 | 万年以上 | 极品～灵丹 | 传说级，往往有高阶妖兽守护 |

### 常见灵草列表

| 名称 | 最低年份要求 | 用途 | 生长环境 |
|:----|:----------:|:----|:--------|
| 筑基草 | 50年可用，百年最佳 | 筑基丹主料 | 灵气充沛的山涧、溪谷 |
| 地髓根 | 百年可用 | 筑基丹主料 | 地下溶洞或灵脉附近 |
| 凝露花 | 十年可用 | 筑基丹辅料 | 清晨多露的山崖 |
| 赤芝 | 百年可用 | 筑基丹辅料 | 阴湿的古木下 |
| 培元果 | 三百年可用，五百年最佳 | 固本培元丹核心主料 | 灵气充沛的山谷深处 |
| 地脉芝 | 百年可用 | 固本培元丹核心主料 | 灵脉节点附近 |
| 归元草 | 十年可用 | 固本培元丹辅料 | 溪流旁，较常见 |
| 寒髓草 | 百年可用 | 多种丹药的调和辅料 | 阴寒之地 |
| 赤阳花 | 百年可用 | 火属性丹药主料 | 火山或地热附近 |

**核心设定**：
- 灵草的年份取决于生长环境的灵气浓度——灵脉越好的地方，灵草长得越快
- 但这意味着好的灵草都长在大宗门的灵脉范围内——散修采药只能去荒山野岭碰运气
- 真正的高年份灵草往往有妖兽守护——灵草和妖兽是共生关系，妖兽守护灵草等成熟后吃掉突破

## 六、丹药成本与交易

### 炼丹成本示例（筑基丹）

| 项目 | 成本 | 说明 |
|:----|:----|:----|
| 筑基草（百年） | 30～50下品灵石 | 核心主料 |
| 地髓根（百年） | 20～40下品灵石 | 核心主料 |
| 凝露花 | 5～10下品灵石 | 辅料 |
| 赤芝（百年） | 15～25下品灵石 | 辅料 |
| 灵泉水 | 1～3下品灵石 | 辅料 |
| 单份材料总成本 | 约70～130下品灵石 | — |
| 正常炼丹出丹率 | 约三成（初级炼丹师） | 三份材料出一枚 |
| 每枚实际成本 | 约210～390下品灵石 | 算上失败成本 |
| 市场售价 | 80～120下品灵石（普通品质） | **算上失败成本其实是亏的** |

**为什么炼丹师还赚钱？**
- 上述计算是按市场价买材料→炼丹→卖成品。真这么干是亏的
- 炼丹师赚钱靠的是：① 自己有渠道低价拿材料（宗门种植或野外自采）；② 炼出中品/上品丹溢价出售；③ 代客炼丹收手续费（材料客户出，成了收丹，废了客户认）
- 林远自己炼丹补根基，不用买成品丹——他出材料和精力，省的是人工溢价

### 丹药交易基本原则

- 黑市交易比正规渠道便宜两到三成——但来历不明，可能是假丹或毒丹
- 大宗门内部丹药有补贴——碧落宫弟子买聚气丹只要市场价的一半
- 散修买丹药只能去黑市或拍卖行——中间商赚差价
- 高品质丹药（上品/极品）不走普通市场——只在拍卖行或熟人圈子流通

## 八、丹毒系统

| 丹毒积累程度 | 症状 | 解决方案 |
|:-----------|:----|:--------|
| 轻度 | 灵力运转稍有不畅 | 停药一个月可自然恢复 |
| 中度 | 修炼效率下降，偶有经脉刺痛 | 需要服用清体丹或找高阶修士帮忙疏导 |
| 重度 | 经脉部分堵塞，修为停滞甚至倒退 | 需要天材地宝洗髓，或自废重修 |
| 极重度 | 丹田被污染，灵力暴走 | 基本无解，非天材地宝不可救 |

**对林远的影响**：
- 林远是杂品灵根，灵力吸纳本就困难——丹毒对纯度的要求更高
- 他长期吃不起高品质丹药，只能用普通品质的——丹毒积累速度比正常修士更快
- 这也是他必须自己炼丹的原因之一：自己炼，可以控制品质，至少保证没有杂质
