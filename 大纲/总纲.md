# 总纲

> 项目：凡仙途｜主角：林远｜题材：玄幻

## 故事一句话
一个杂灵根的凡人少年，因一块碎玉全村被灭，在追杀与逃亡中被迫踏上修仙之路——复仇只是起点，长生的路比报仇远得多。

## 创意约束
- **反套路规则**：反派智商在线，主角赢一次要付大代价；没有"捡到神功秒变强者"的桥段
- **硬约束（世界/能力/行为）**：凡人王朝禁修仙；大境界压制不可逾越；碎玉每次突破会暴露位置
- **主角缺陷**：出身卑微信任人，认知局限易被骗，前期笨拙地学习世界规则
- **反派镜像**：玄冥宗不是纯粹的恶——在他们看来灭口是为了保密。正魔两道没有绝对好坏，只有利益

## 核心设计理念：代价法则
> 这个世界的底层规则——**没有免费的东西，所有获得都对应付出。**

- **碎玉不是外挂**：它只是磨平了杂灵根和正常修士的起跑线差距，不是让林远抄近道。该受的伤、该吃的苦一点不会少
- **收获必有代价**：获得功法→时间精力+选择正确路径；拿到灵药→消耗战力储备+受伤；打赢战斗→重伤+底牌尽出
- **用心做事自然有回报**：林远得到不是因为他运气好，而是因为他比别人更认真、更坚持——比如别人随便扫扫藏书阁，他认真打扫所以发现了角落的功法
- **选择产生歧路**：每个关键收获节点，林远都面临选择。读者会讨论"如果当时选另一边会怎样"
- **禁止白送**：任何机缘都不能靠运气撞见、老怪物指引、天降馅饼获得

## 宿命之敌·楚玄

> 不是反派，是镜像——和林远同辈的平行成长型宿敌。两人起点相似（杂灵根），选择了不同的路，各自走到了不同的终点。读者会争论"林远换成楚玄的处境会怎么做"。

### 基础档案
| 项目 | 内容 |
|:----|:-----|
| 全名 | 楚玄 |
| 出场 | 第1卷·Ch38 废弃矿洞初遇 |
| 身份 | 没落修仙家族最后一人，楚家灭门唯二幸存者（带幼妹逃亡但最终失去） |
| 灵根 | 杂品灵根（与林远同层级） |
| 外貌 | 高瘦，眼神锐利带刺，右眉一道竖疤（小时被妖兽抓伤） |
| 性格 | 果断张扬，不相信任何人，行事风格比林远更狠——"我宁愿错杀，不放过" |
| 核心执念 | 变强是唯一目的，其他都是手段 |

### 镜像对照：林远 vs 楚玄
| 维度 | 林远 | 楚玄 |
|:----|:----|:-----|
| 出身 | 凡人村落，灭村后被逼修仙 | 没落修仙家族，从小知道修仙界规则 |
| 初始性格 | 谨慎隐忍，习惯先观察再行动 | 果断张扬，敢赌敢拼，先做了再说 |
| 行事风格 | 能跑就不打，打就要赢 | 能打就不跑，跑是为了回来打 |
| 对他人态度 | 愿意相信人，被骗了才学会警惕 | 默认所有人不可信，极少数人例外 |
| 驱动力 | 复仇是起点，逐渐找到自己的道 | 变强是唯一目的——"弱就是原罪" |

### 三阶段演进
| 阶段 | 卷 | 事件 | 关系状态 |
|:----|:---|:-----|:---------|
| ① 初遇 | 第1卷·Ch38 | 废弃矿洞入口争夺，各自忌惮退去 | 陌生人→记住彼此，知道对方不是软柿子 |
| ② 重逢 | 第2卷·Ch65.6 | 落霞集溪边再遇，组队探遗迹 | 短暂合作→各取所需→分道扬镳 |
| ③ 各自的路 | 第3卷+ | 天罡宗 vs 碧落宫，正魔两道不同阵营 | 远期碰撞锚点——未来必有一战或一次关键选择 |

### 与剧情的关系
- **Vol1-2**：背景板存在，只在关键节点出现，制造"另一个杂灵根走到哪了"的参照
- **Vol3+**：进入天罡宗，成为林远在正道的对应面。读者会不自觉比较两人的进度和选择
- 楚玄**不是每章都出现**的反派，而是阶段性出现的"镜子"——每次出现都让读者重新审视林远的选择

## 法宝·功法体系（前三卷）

## 核心主线
- **主线目标**：追查灭村真相 → 复仇 → 在追杀中活下去 → 追求长生之路
- **主要阻力**：玄冥宗的追杀（外部）、资质低劣（自身）、不了解修仙世界（认知）

## 核心暗线
- **暗线主题**：一个善良的人如何在残酷世界中保持本心，而不是变成自己仇恨的人
- **回收节点**：每当林远面临"要不要为了变强去做违心的事"时，暗线浮现

## 世界观/力量体系简介
- **世界观要点**：大夏王朝（凡人界禁修） → 越国修仙界（散修边缘） → 中州（三足鼎立，弱肉强食）——世界逐步扩张，不是主角主动探索，是被追杀逼着跑出去
- **力量体系要点**：练气→筑基→金丹→元婴→化神，每阶分初/中/后/巅峰。杂灵根修炼极慢，碎玉加速但会暴露

## 主角成长线
- **起点状态**：15岁凡人村童，杂灵根，对修仙世界一无所知
- **关键跃迁节点**：
  - 灭村之夜 → 入世（练气初期）
  - 第一次反杀 → 理解修仙的残酷（练气中期）
  - 一枚筑基丹不够，碎玉主动共鸣补足药力成功突破 → 灵力波动暴露位置成为新枷锁（筑基初期）
  - 被迫叛逃宗门 → 彻底放弃幻想，只信自己（筑基初期，根基正常但有轻微裂痕）
- **终局定位**：化神初期——没有追求力量的极致，但找到了自己的道心。他不是"最强的人"，他是"最清醒的人"

- **第90-100章：墨渊设局坑林远——明知遗迹有埋伏不告诉他，但林远阴差阳错发现密室拿到意外收获；林远主动递出功法残篇，墨渊第一次"用新眼光看他"**
- 第95章：反杀筑基初期执事，坐尸体旁掏出筑基丹看了一眼——"快了"
- 第100章：卷末——练气巅峰，身怀筑基丹，但被结丹长老盯上

### 第三卷
- 第102章：玄冥宗结丹长老周鹤鸣亲自出手，追杀全面升级
- 第108-109章：绝境中决定冲击筑基，服下筑基丹——药力逸散，碎玉主动共鸣补足
- 第110章：碎玉补足药力成功突破，筑基初期（根基正常），但灵力波动暴露位置，决心去碧落宫避风头
- 第117章：向苏青璃试探炼丹权限，发现杂役身份碰不到丹炉
- 第124章：在藏经阁发现《筑基丹解》残本——"余以杂灵根筑基，耗四枚筑基丹"
- 第126章：宗门大比被迫出手，压制修为演戏
- 第130章：玄冥宗使者来碧落宫，危机预感
- 第138-140章：发现碧落宫要交人，趁夜摸药材库+藏经阁第三层偷密道地图
- 第143-147章：叛逃被堵截，以筑基初期的修为硬战筑基中期，碎玉全力催动暴露位置
- 第150章：卷末——筑基初期（根基正常，丹田有轻微裂痕），被周鹤鸣私追+陈长老通缉

### 第五卷
