# 章节审查清单（6 维）

> 写完一章后，让 AI 按以下维度逐项检查。
> 用法：粘贴正文后说 **"按审查清单检查"**

---

## 一、设定一致性（setting）

- [ ] 角色能力是否与当前境界匹配（力量体系.md）
- [ ] 角色装备/法宝是否符合境界获取条件（主角装备路线.md）
- [ ] 碎玉能力使用是否符合规则（灵韵共鸣被动触发，不主动；缓慢温养以年为单位）
- [ ] 地点描述是否与世界观一致（五大洲/宗门/城市/秘境）
- [ ] 功法/术法名称是否来自设定集，无自创
- [ ] 货币/资源体系是否符合已建立的规则

## 二、时间线（timeline）

- [ ] 本章时间是否与上章衔接（无回跳或无合理解释的时间跳跃）
- [ ] 卷纲中规定的日期/倒计时是否正确推进
- [ ] 角色未同时出现在两个地点
- [ ] 季节/天象与章节情绪匹配（非强制，但注意明显矛盾）

## 三、叙事连贯（continuity）

- [ ] 上章章末钩子在本章有回应
- [ ] 场景转换有合理过渡
- [ ] 情绪弧连续（上章愤怒→本章平静需有过渡）
- [ ] 时间跳跃处有交代（"三天后""半月过去"）

## 四、角色一致性（character）

- [ ] 对话风格符合角色设定（林远/配角各有特征，不全员书面语）
- [ ] 行为与已建立的性格/动机一致
- [ ] 角色未使用"不应知道的信息"
- [ ] 角色之间的关系符合设定（兄弟/仇敌/师徒等）
- [ ] 林远的行为模式：谨慎/隐忍/务实，不突然莽撞或话多

## 五、逻辑（logic）

- [ ] 因果关系成立（行动有动机，结果有原因）
- [ ] 战斗/冲突结果符合已建立的力量对比
- [ ] 角色决策有合理动机，不为推动剧情而强行降智
- [ ] 追逐/逃逸场景的时空逻辑合理（距离感清晰）

## 六、AI味检查（ai_flavor）

### 6.1 词汇层
- [ ] 无高频 AI 词汇密集出现（每个类型每 500 字 ≤ 1 次）：
  - 副词："缓缓/淡淡/微微/轻轻"+动词
  - 神态模板："眸中闪过""瞳孔微缩""心中一凛"
  - 递进模板："更重要的是""值得一提的是""值得注意的是"
- [ ] 无总结词：综合/总之/由此可见/总而言之

### 6.2 句式层
- [ ] 无"起因→经过→结果→感悟"四段闭环（删最后感悟句）
- [ ] 无连续 3 句以上同构句（主谓宾结构一致）
- [ ] 段末不以总结句收尾（"他终于明白了""由此可见"）
- [ ] 无同一信息用不同句式重复 2-3 遍

### 6.3 叙事层
- [ ] 节奏有疏密对比（非匀速，有的段落仅一句话）
- [ ] 无"他不知道的是……""殊不知……"戏剧性反讽提示
- [ ] 章末不"安全着陆"——至少留下一个未解决的问题
- [ ] 展示后不紧跟解释（用了动作展示，不补解释句）

### 6.4 情感层
- [ ] 情绪通过生理反应+微动作展示，非"他感到愤怒"式标签
- [ ] 情绪切换有过渡（上句愤怒→下句平静需中间句）
- [ ] 不同角色有不同反应模板（非全员"瞳孔微缩""心中一凛"）

### 6.5 对话层
- [ ] 对话在推进冲突/塑造角色，非解释背景
- [ ] 角色各有口语特征（有人啰嗦/有人惜字如金/有口头禅）
- [ ] 对白后无解释性叙述（"他这么说是因为……"）
- [ ] 有抢话/沉默/答非所问等真实对话特征
- [ ] said tag ≤ 30%，优先使用前置动作替代

---

## 输出格式要求

审查完成后，AI 按以下格式输出：

```
## 审查结果

### 阻塞（blocking）
- [问题描述+证据位置]

### 高优
- [问题描述+证据位置]

### 中低
- [问题描述+证据位置]

---
总结：N 个问题，X 个阻塞，Y 个高优
```

**对比 reviewer.md 的原始 6 维审查标准**：本清单已针对《凡仙途》项目适配，增加了碎玉能力规则、林远行为模式等专属检查项。
